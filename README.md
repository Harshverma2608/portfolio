# Harsh Verma — Portfolio

## Quick Start

### 1. Setup server env
```
cp server/.env.example server/.env
# Fill in MONGO_URI and other values
```

### 2. Run backend
```
cd server && npm run dev
```

### 3. Run frontend (separate terminal)
```
cd client && npm run dev
```

Frontend → http://localhost:5173  
Backend  → http://localhost:5000

## Adding your images

Save these files exactly as named:

```
client/public/avatar.jpg              ← your personal photo
client/public/projects/foodster.png  ← Foodster screenshot
client/public/projects/contentbroadcast.png  ← ContentBroadcast screenshot
client/public/projects/aamovies.png  ← A&A Movies screenshot
```

Take a screenshot of each live site and save them there. Images are already wired up — just drop the files in.
